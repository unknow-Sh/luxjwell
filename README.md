# Luxe Jewels — Jewelry E‑commerce (React)

A fully animated React jewelry e‑commerce site with **New Rocker** (Impallari Type) for headings, a custom color shader theme, and static add‑to‑cart.

## Features

- **Home**: Hero, 4 main categories (Rings, Necklaces, Earrings, Bracelets), quote block, featured products
- **Shop**: Filter by category and price, sort (featured, price, rating), 18 static jewelry products
- **Product detail**: Rating, reviews count, quantity, add to cart, similar products
- **Cart**: Update quantity, remove items, static summary and link to checkout
- **Checkout**: Static form (no real payment)
- **About** & **Contact**: Content and contact form (static)
- **Animations**: Framer Motion for page transitions, hover, stagger, and scroll-triggered animations
- **Theme**: Dark palette with gold/rose accents and gradient shaders

## Run

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```

## Tech

- React 19 + Vite 7
- React Router 7
- Framer Motion 12
- Google Fonts: New Rocker, Cormorant Garamond
